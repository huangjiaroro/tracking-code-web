# 快速安装指南

## 一键安装

扩展文件已准备就绪！图标已自动生成，现在可以直接安装。

## 安装步骤

### 1. 打开 Chrome 扩展管理页面

在 Chrome 地址栏输入：
```
chrome://extensions/
```

### 2. 启用开发者模式

在页面右上角，打开"开发者模式"开关

### 3. 加载扩展

1. 点击"加载已解压的扩展程序"按钮
2. 在文件选择器中，导航到以下文件夹：
   ```
   /Users/zouying/clawd/skills/ui-marker/chrome-extension
   ```
3. 点击"选择文件夹"

### 4. 完成安装

✅ 如果看到 "UI Marker" 扩展出现在列表中，说明安装成功！

## 使用方法

1. **打开任何网页**（建议先试试简单的页面）
2. **点击工具栏上的 UI Marker 图标**（蓝色靶心图标）
3. **点击"✨ 开始标记"按钮**
4. **查看标记结果**

## 查看详细效果

标记完成后：
- 页面上的可交互元素会显示洋红色边框
- 弹出窗口会显示详细的统计信息
- 按下 `F12` 打开浏览器控制台，可以看到更详细的分析日志

## 清除标记

点击弹出窗口中的"🧹 清除标记"按钮即可。

## 遇到问题？

### 扩展未出现？
- 确认选择了正确的文件夹：`/Users/zouying/clawd/skills/ui-marker/chrome-extension`
- 检查扩展页面是否有错误提示（红色文字）

### 无法标记某些页面？
- 某些特殊页面（如 chrome:// 开头的页面）受安全限制，无法标记
- 刷新页面后再试

### 标记影响页面使用？
- 标记只是添加了视觉效果，不影响功能
- 随时可以点击"清除标记"恢复正常
